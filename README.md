## Description

Full-stack app with Google auth for ordering food and receiving order via email.

Also check a [mobile application](https://github.com/Yevhenns/pizza_mob.git) and [back-end application](https://github.com/Yevhenns/pizza-nest-pg.git) for this project.

## Installation

Copy .env.example, rename it to .env.local and fill the environment variables.

```bash
npm i
```
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the
result.
